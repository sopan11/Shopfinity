Shopfinity

Shopfinity is a Django-based e-commerce web app that allows users to browse products, add them to a cart, and place orders through a simulated payment gateway. It supports user authentication, dynamic cart handling with localStorage, and secure checkout with CSRF protection. Orders are linked to logged-in users, capturing essential details like name, address, and email. The project showcases Django’s strengths in building secure, interactive, and scalable shopping platforms.
