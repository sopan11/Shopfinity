About Shopfinity

Shopfinity is a simple and modern e-commerce web application built using Django. It allows users to explore products, manage a cart, and securely place orders. Designed as a full-stack project, it features dynamic cart functionality using JavaScript and `localStorage`, user authentication, order tracking, and a simulated payment system. Shopfinity is ideal for learning and demonstrating core web development concepts with Django.
